# 🔐 Password Generator

A simple Python project that generates strong and secure passwords using random combinations of letters, numbers, and symbols.

## 🚀 Features
- Generates random passwords
- Customizable length
- Uses letters, digits, and special characters
- Beginner-friendly Python project

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Install Python on your system
2. Clone this repository
3. Open the folder in VS Code
4. Run the file:
   python password_generator.py

## 📌 Example
Generated Password: A@7kL9#pQ2

## 📚 Learning Purpose
This project is made for learning Python basics like loops, random module, and string handling.
